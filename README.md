## что делает в первую очередь наш?
в первую очередь это чат на весь экран или примерно 80% ~ 65% от всего
пространства экрана.

любой пользователь имеет 3 вещи:
- имя
- id
- сообщения к которым он превязан тоесть любое сообщение имеет  id отправителя и
  инкрементное id от первого сообщения пользователья и id++ .

** уточнение: это примерные заресовки то го что будет.


чат делится на комнаты на общий и на личные.

на первых парах пользователь не будет точно иметь систему авторизации и будет
иметь только имя, id будет создаватся автоматически.
