# что делает программа
главная идея программы это удобный (в моем понимании) чат который можно легко запустить на debian или
его производном дистробутиве через один скрипт который будет работать через локальную сеть тоесть по  wifi
или по проводу через маршутизатор при желании можно запустить на каком нибудь  vps но при желании.

тоесть локальный чат через браузер.

в этом проекте расматривается использование fastapi  для бекенда и javascript как языка фронтденда.


тоесть проще говоря попытка удобный чат через браузер и стек технологий которые мне новы.
